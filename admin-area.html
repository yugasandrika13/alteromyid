API Sistem

Pengaturan Moota sekarang disederhanakan menjadi 2 field wajib: Webhook URL HTTPS dan Secret Token untuk verifikasi signature HMAC.

Gunakan URL webhook publik dari Cloudflare Worker atau proxy Anda. Jangan gunakan URL Google Apps Script langsung.

Catatan penggunaan

1. Moota mengirim payload JSON array via `POST` dan signature di header `Signature`.

2. Jika Anda memakai Google Apps Script, arahkan webhook ke endpoint Cloudflare Worker atau proxy publik terlebih dahulu agar header `Signature` bisa diteruskan ke Apps Script.

3. Jalankan `Test Koneksi Moota` sebelum menyimpan agar Webhook URL dan Secret Token tervalidasi lebih dulu.

Isi public key, URL endpoint, dan private key langsung dari aplikasi. Private key hanya disimpan di server.
